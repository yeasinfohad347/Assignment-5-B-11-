Question-1: What are the different ways to select an element in the DOM?

Here are several ways to select elements in the DOM using JavaScript. The getElementById() method selects an element by its unique ID. The getElementsByClassName() and getElementsByTagName() methods select elements by class name or tag name, respectively. The querySelector() and querySelectorAll() methods allow you to select the first matching element or all matching elements using CSS selectors

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML retrieves or sets the HTML content of an element, including any HTML tags inside it. innerText retrieves or sets the visible text content of an element, excluding HTML tags and respecting CSS styles like display: none. textContent retrieves or sets the text content of an element, including hidden text, but excludes HTML tags. While innerHTML can be used to manipulate HTML structure, innerText and textContent are typically used for working with plain text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where a single event listener is attached to a parent element instead of individual child elements. It leverages the event bubbling mechanism, where events propagate up from the target element to its ancestors. By checking the event.target, we can determine which child element triggered the event. This improves performance and reduces the need to add multiple event listeners for dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process in the DOM where an event starts at the target element and then propagates upward through its ancestors in the DOM hierarchy. When an event is triggered, it first runs on the innermost element (the target), and then it moves to the parent element, then the grandparent, and so on, until it reaches the root of the document.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element in JavaScript, use document.createElement(tagName). To add the element to the DOM, we can use appendChild() on a parent element. To remove an element, you can use remove() on the element itself or removeChild() on its parent. These methods allow you to dynamically modify the HTML structure of our page.
